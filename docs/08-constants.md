# 08. 定数リファレンス

すべてのバランス数値は `src/data/constants.ts` に集約（ロジックにハードコードしない方針）。
演出の定数は各描画ファイルにある（§末尾）。

---

## FIELD（フィールド・物理）

| 定数 | 値 | 意味 |
|---|---|---|
| `rField` | 30 | 場外境界（原点からこの半径外は場外＝暴発/消滅） |
| `sMax` | 5 | 属性強度の最大値 |
| `zPeak` | 5 | 強度ピーク V（|z|=zPeak で強度最大の山型） |
| `epsilon` | 0.35 | 中立しきい（\|z\|<ε は中立） |
| `aMax` | 13 | 加速度上限（\|z\|≈0 で最大加速・中立が最速） |
| `zRef` | 2.5 | 加速度基準（\|z\|=zRef で加速度 0、超えると減速） |
| `aDecelMax` | 8 | 減速の上限（負の加速度の頭打ち） |
| `dt` | 1/60 | 積分ステップ（秒） |
| `fixedSpeed` | 10 | 初速（毎回固定） |
| `minSpeed` / `maxSpeed` | 4 / 14 | 互換用の安全境界（UI スライダーは廃止） |
| `maxFlightSpeed` | 24 | 飛行速度の上限（終端速度） |
| `aoeRadius` | 5 | 暴発の AoE 半径 |

> 設計の肝：`zRef(2.5) < zPeak(5)`。最強属性を狙うほど減速し、速度 0 で霧散する精度ゲームになる。

| 定数 | 値 | 意味 |

| `orbitWallCarveRadius` | 0.5 | 周回が壁に触れて 1 点でえぐる半径 |
| `orbitHealScale` | 5 | （旧モデル）光の周回回復＝強度 × これ |

| `flinchBaseTurns` | 1 | ひるみ（光）の基本ターン数 |

| `burnScale` | 2 | 継続ダメージ総量 = \|z\| × これ を burnTurns に分割 |

## RUPTOR（崩し手・#42／[05b-enemy-archetypes.md](05b-enemy-archetypes.md) §4）

z 場に極（1/x 型の特異点）を仕込み、狙点で暴発させる敵の z 場パラメータ。
`z(x,y) = zBase×極性 + poleK/g(x,y)`（`g` は狙点で 0 になる接近方向の符号付き距離）。

| 定数 | 値 | 意味 |
|---|---|---|
| `zBase` | 2.2 | 飛行区間の基礎 z（zRef 以下＝自然失速しない範囲で、属性がはっきり読める強さ） |
| `poleK` | 2 | 極の強さ係数。大きいほど手前から z が急峻になり、減速域が広がる（迎撃猶予は増える） |

## INSTABILITY（暴発の不安定化・累積・崩壊・04b）

「膜の摩耗」＝ラン全体（全7面通し）で累積するカウンター。暴発が解決するたび +1（味方でも敵でも同じ）。

| 定数 | 値 | 意味 |
|---|---|---|
| `firstCollapseThreshold` | 5 | この暴発回数で初回崩壊（グリモワールの介入・一度きり） |
| `firstCollapseStage` | 6 | 初回崩壊の保証面（1始まり）。遅くともこの面のターン2以降で必ず発生 |
| `misfireLimit` | 12 | 致死しきい：この回数で破局（ステージ全体暴発＝ゲームオーバー・救済なし） |
| `vStart` | 2 | 半径ばらつきの開始 count（これ未満はブレなし） |
| `vMax` | 0.6 | 半径ばらつきの最大（実半径 = aoeRadius×(1±v)） |
| `relief` | 1 | 暴発ゼロでステージクリア時の緩和（−1・下限0） |
